# now-worldcup2026-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及 2026 年世界杯相关主题，包含信息展示、资料整理、工具页面等多种类型。

## 项目简介

`now-worldcup2026-hub` 是一个静态 HTML 页面集合仓库。所有页面均为独立文件，不依赖外部后端服务，可直接通过浏览器打开或部署至任意静态托管平台。仓库本身不针对特定域名或网站，仅作为内容归档与发布的统一入口。

## 目录说明

```
/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 主要页面存放目录
│   ├── page-xxx.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，文件名通常反映页面主题。
- `assets/`：页面引用的共用样式、脚本及图片资源，按类型分目录存放。
